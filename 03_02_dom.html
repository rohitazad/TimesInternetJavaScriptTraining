JavaScript DOM: Overview and Examples

1. What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects, allowing languages like JavaScript to interact with the web page.

// Example: Accessing the DOM
document.body.style.backgroundColor = "lightblue";

// Example: Changing the text of an element
document.getElementById("demo").innerHTML = "Hello, World!";

2. Accessing DOM Elements

JavaScript provides several methods to access DOM elements. Some common methods include getElementById, getElementsByClassName, getElementsByTagName, and querySelector.

// Example: Accessing an element by ID
let elementById = document.getElementById("myId");
console.log(elementById);

// Example: Accessing elements by class name
let elementsByClassName = document.getElementsByClassName("myClass");
console.log(elementsByClassName);

// Example: Accessing elements by tag name
let elementsByTagName = document.getElementsByTagName("p");
console.log(elementsByTagName);

// Example: Accessing an element using querySelector
let elementByQuerySelector = document.querySelector(".myClass");
console.log(elementByQuerySelector);

3. Modifying DOM Elements

You can modify the content and attributes of DOM elements using JavaScript. Common operations include changing innerHTML, setting attributes, and manipulating styles.

// Example: Changing innerHTML
document.getElementById("myId").innerHTML = "New Content";

// Example: Setting an attribute
document.getElementById("myId").setAttribute("class", "newClass");

// Example: Modifying styles
document.getElementById("myId").style.color = "red";
document.getElementById("myId").style.fontSize = "20px";

4. Creating and Removing DOM Elements

You can create new DOM elements and remove existing ones using JavaScript. This is useful for dynamically updating the content of a web page.

// Example: Creating a new element
let newElement = document.createElement("div");
newElement.innerHTML = "This is a new element";
document.body.appendChild(newElement);

// Example: Removing an existing element
let elementToRemove = document.getElementById("myId");
elementToRemove.parentNode.removeChild(elementToRemove);

5. Event Handling

JavaScript allows you to handle events such as clicks, mouseovers, and key presses. You can add event listeners to DOM elements to execute code in response to these events.

// Example: Adding a click event listener
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

// Example: Adding a mouseover event listener
document.getElementById("myDiv").addEventListener("mouseover", function() {
    this.style.backgroundColor = "yellow";
});

6. Practical Examples

Let's see some practical examples of DOM manipulation in action. These examples demonstrate common tasks like changing content, adding elements, and handling events.

// Example: Changing content on button click
document.getElementById("changeContentButton").addEventListener("click", function() {
    document.getElementById("content").innerHTML = "Content has been changed!";
});

// Example: Adding a new list item on button click
document.getElementById("addItemButton").addEventListener("click", function() {
    let newItem = document.createElement("li");
    newItem.innerHTML = "New Item";
    document.getElementById("myList").appendChild(newItem);
});

// Example: Removing an item on button click
document.getElementById("removeItemButton").addEventListener("click", function() {
    let list = document.getElementById("myList");
    list.removeChild(list.lastElementChild);
});
